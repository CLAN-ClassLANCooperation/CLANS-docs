# Class LAN Drop 教室端（CR-Side）部署

::: caution 不兼容性提醒
CLAND 系列产品仅支持 **Windows 10/11** 操作系统，
没有也不会有针对 Windows 8.1/7 的旧版本 Windows 的兼容更新推送，
且不兼容 x86 架构系统。

如有需要，您可以 Fork 一份项目仓库并 **自行开发** 针对旧版/X86架构的 Windows 或 macOS/Linux 的兼容版本。
但请注意，这些社区成员制作的版本不会受到 issue 支持。也就是说，如果您在使用这些版本中遇到了问题，请不要在原仓库提 issue ！
:::

## 下载

您可在 CLAND 教室端（以下简称 `CLANDCR` ）的 [GitHub Release](https://github.com/CLAN-ClassLANCooperation/ClassLanDrop-CRSide/releases) 获取 CLANDCR 的最新二进制构建。
